## UTNG Recruiting App

### Requirements
- `ruby 3.0.0.`
- `rails 6.1.4`
- `postgresql 13.3`

### Install
```
cd /into/desired/path
git clone git@github.com:andykallhoff/utng_leads.git
cd utgn_leads
bundle install
rails db:create
rails db:migrate
rails s
```
